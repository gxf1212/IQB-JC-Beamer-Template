# IQB Journal Club Beamer Template Tools

Utility scripts for working with the IQB Journal Club Beamer template.

## Quick Usage

For detailed documentation, installation instructions, and examples, see the comprehensive guide in `software-copyright/3-basic-usage.tex`.

## Available Tools

### 1. `extract_pdf_page.py`
Extract PDF pages as PNG for visual analysis.
- Documentation: `software-copyright/3-basic-usage.tex` (section \ref{sec:extract-pdf-page})

### 2. `pdf_to_pptx.py`
Convert PDF to PPTX (image-based, view-only).
- Documentation: `software-copyright/3-basic-usage.tex` (section \ref{sec:pdf-to-pptx})

### 3. `pdf_to_editable_pptx.py`
Convert PDF to editable PPTX with perfect layout preservation.
- Documentation: `software-copyright/3-basic-usage.tex` (section \ref{sec:editable-pptx})
